# README - Десктопная версия приложения для автоматизации лабораторных работ по физике в университете

Используется для работы с лабораторными работами без интернет соединения

- Vue.js 3 
- Node.js 
- Electron (Автоматическое преобразование Vue проекта в установщик десктопного)
